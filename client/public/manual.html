AetherHome Z3000 User Manual

1. Introduction

The AetherHome Z3000 is a next-generation smart home hub designed to bring complete automation to residential spaces. From lighting and temperature control to security and entertainment, this device allows seamless integration and control via a central system or mobile application.

2. Key Features

• Supports over 500 smart devices
• Voice control compatibility with Alexa and Google Assistant
• Built-in AI-driven automation suggestions
• Advanced user profiles and permissions
• Encrypted data communication for enhanced security

3. System Requirements

• Dual-band Wi-Fi (2.4GHz and 5GHz)
• Android 8.0+ or iOS 13.0+ for mobile app
• AetherHome App (available on Play Store and App Store)
• Minimum 1 Mbps constant internet speed

4. Unboxing and Components

Inside the AetherHome Z3000 package, you will find:
• AetherHome Z3000 controller unit
• Power adapter
• Wall mount kit
• Quick start guide
• Warranty card

5. Installation Guide

Mount the AetherHome Z3000 on a wall or place it on a flat surface. Connect the power adapter and wait for the LED to turn blue. Using the AetherHome app, follow the setup wizard to connect the device to your network.

6. Mobile App Setup

Download the app, create an account, and follow the prompts to connect your AetherHome Z3000. Use the QR code on the device or enter the serial number manually.

7. Adding Smart Devices

Tap 'Add Device' in the app. Ensure the target device is in pairing mode. Follow the specific prompts for each device type. Successful pairing will be indicated by a confirmation message and icon in the dashboard.

8. Voice Assistant Integration

Navigate to the 'Voice Assistant' section in the app. Select your preferred platform (Alexa/Google), log in, and authorize access. You can now control devices with voice commands.

9. Creating Automations

Automations are rules that trigger actions. Example: Turn off all lights when 'Away Mode' is activated. Automations can be based on time, location, device status, or custom inputs.

10. Scenes and Profiles

Scenes are predefined settings like 'Movie Night' or 'Morning Routine'. Profiles allow different users to have customized access and controls.

11. Security Features

Includes biometric app access, 2FA, and local/offline control fallback. All device data is encrypted end-to-end.

12. Maintenance Tips

• Keep firmware updated
• Use only the original power adapter
• Clean device exterior with a dry cloth
• Reboot once a month for optimal performance

13. Troubleshooting

• Device not showing online? Reboot router and device.
• Smart device unresponsive? Re-pair using the app.
• Contact support if issues persist beyond basic troubleshooting.

14. FAQs

Q: Can I use it without internet?
A: Basic local control works, but cloud functions need internet.

Q: How many devices can I connect?
A: Up to 512 supported.

Q: Can guests control my devices?
A: Yes, using guest profiles.

15. Regulatory Compliance

The AetherHome Z3000 complies with FCC, CE, and RoHS standards. Do not open the device casing, as it may void your warranty.
